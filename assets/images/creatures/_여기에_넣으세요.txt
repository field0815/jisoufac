[ 실장석(크리처) 스프라이트 시트를 여기에 넣으세요 ]

★ 모든 실장석은 "4프레임 × 4방향" 스프라이트 시트로 움직입니다.
   파일 1개 = 가로 4칸(프레임) × 세로 4줄(방향) 격자.

   ┌────┬────┬────┬────┐  ← 열 = 애니메이션 프레임 0,1,2,3
   │ 앞 (아래로 볼 때)   │  1행
   ├────┼────┼────┼────┤
   │ 좌 (왼쪽 이동)      │  2행
   ├────┼────┼────┼────┤
   │ 우 (오른쪽 이동)    │  3행
   ├────┼────┼────┼────┤
   │ 뒤 (위로 이동)      │  4행
   └────┴────┴────┴────┘

파일명(그대로):
  adult.png       ← 성체실장      maggot.png ← 구더기
  thumb.png       ← 엄지          child.png  ← 자실장
  slime.png       ← 점액 덩어리(출산 직후, 10초 뒤 구더기)
  pet_adult.png   ← 성체 사육실장  pet_child.png   ← 새끼 사육실장
  slave_adult.png ← 독라(성체)     slave_child.png ← 새끼 독라

* 표시 크기: 자실장은 2/3, 엄지·구더기·점액은 1/2로 자동 축소됩니다(겹치면 아래쪽이 위로).

* 한 칸 크기는 자유 — 게임이 이미지를 4×4로 균등 분할해 사용(예: 한 칸 48x48 → 전체 192x192).
* 투명 PNG 권장. 파일 없으면 살짝 들썩이는 색 동그라미 플레이스홀더.
* 프레임 속도: js/config.js 의 ANIM_FPS(기본 6).
* 파일명 변경은 js/config.js 의 G.CREATURES.img.
