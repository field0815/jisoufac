# Tier 3 Building Images

- `large_warehouse.png`: 대형 비축고. 권장 192x192 PNG, 정적 1프레임.
- `chaos_turret.png`: 카오스 포탑. 권장 384x96 PNG.
  - 96x96 프레임 4장을 가로로 배치합니다.
  - 게임에서 4프레임 애니메이션으로 반복 재생합니다.
- `chaos_gate.png`: 카오스 게이트. 정확히 192x48 PNG.
  - 48x48 프레임 4장을 가로로 배치합니다.
  - 프레임 순서는 대기부터 활성 상태까지 자유롭게 구성할 수 있습니다.
  - 원본 방향은 오른쪽이며 게임에서 입구-출구 방향에 맞춰 회전합니다.

파일이 없을 때는 건물별 지정 색상의 플레이스홀더가 표시됩니다.
