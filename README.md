# 실장석 공장 (Siljangseok Factory)

쿠키클리커 + 팩토리오 스타일 **물류 · 운송 · 사육 · 제작 · 판매** 게임.
주인공은 육류 판매업자. 실장석을 포획·번식·사육·가공·판매하여 돈을 번다.

## 실행 방법
`index.html` 더블클릭 또는 브라우저로 열기. (해상도 1440×960, 창에 맞춰 자동 축소) 시작 자금 ₩5,000.
※ `D:\OneDrive\게임제작\삼도호텔` 은 별개의 RPG Maker 프로젝트이니 혼동 주의.

## 폴더 구조
```
실장석공장/
├─ index.html
├─ css/style.css
├─ js/
│   ├─ config.js     ← 모든 수치/장치/가격계수/에셋 경로 (밸런스 조절 지점)
│   ├─ assets.js     ← 이미지/효과음 로더 (없으면 색상 플레이스홀더)
│   ├─ state.js      ← 게임 상태
│   ├─ creatures.js  ← 스탯/번식/성장/가격/생산품
│   ├─ pens.js       ← 우리(사료 소모·운치 배설·성장·수용량)
│   ├─ park.js       ← 공원(포획)
│   ├─ pen.js        ← 공장 좌측 "우리 도크"(성체 드래그 투입)
│   ├─ factory.js    ← 공장(그리드/벨트/장치/시뮬레이션) ★핵심
│   ├─ ui.js         ← 상단바 + 오버레이(거래/연구/통계) + 옵션/저장
│   └─ main.js       ← 초기화 + 게임 루프
└─ assets/images/{creatures,devices,backgrounds,ui}/ , assets/sounds/{sfx,bgm}/
```

## 그래픽 / 효과음 (스프라이트 시트만 넣으면 적용)
모든 그래픽은 **4프레임 애니메이션**으로 돌아갑니다. 각 `assets/...` 폴더의 `_여기에_넣으세요.txt`에 격자 규격이 있습니다.
- **실장석**: `creatures/` → adult/maggot/thumb/child .png — **4프레임 × 4방향(앞/좌/우/뒤)** = 4열×4행 시트. 이동 방향에 맞는 줄이 자동 선택됨.
- **장치**: `devices/` → belt, crossbelt, tunnel, sorter, grabber_rail/**grabber**(집게/긴팔 집게 레일+헤드), warehouse, packer, penbox_tile/**penbox**(바닥+울타리), birthing/**birthing_ready**(빈 상태), washbasin, slaughter, deshell, grinder, correction/**correction_teacher**(교사 장착), mixer, cookery, 특수(speaker/pointer/catcher/skewer_dev/**skewer_dev_loaded**/feeder/**turret**) .png — 대부분 **가로 4프레임** 시트.
- **화물(생산품/자원)**: `products/` → meat(실장육)/minced(분쇄육)/food(실장푸드)/unchi(운치)/**bezoar_s·bezoar_m·bezoar_l(소·중·대형 위석)**/seasoning(조미료)/jisso_acid(짓소산)/jisso_food(짓소산 푸드)/skewer(꼬치훈제)/can(통조림)/vacuum(진공포장)/jeotgal(실장젓갈)/muchim(실장무침) .png — **정적 1장(애니메이션 없음)**. 아이콘이 곧 맵 위 화물 그래픽.
- **배경**: `backgrounds/` → **factory.png**(공장 화면 고정 배경, 1152×720 권장), **tile.png**(공장 바닥 타일 — 칸(48px)마다 반복, 소유 영역에만 깔림, 정사각 48/96/144… 권장), **park.png**(공원 배경 — 필드 전체 cover, **1440×786 권장**).
- 효과음: `sounds/sfx/` → capture, place, rotate, birth, wash, sell, click, remove, grow .mp3
- 프레임 속도는 `js/config.js`의 `ANIM_FPS`(기본 6). 이미지가 없으면 살짝 움직이는 색 플레이스홀더로 표시됩니다.

## 핵심 시스템
- **체력/행복 스탯**: 모든 실장석은 **체력(최대=크기)**·**행복(0~100)**을 가짐(클릭 정보창에서만 표시). 포탑/노동석/침입 전투 및 교정시설이 이 체력을 깎고, 체력 0이면 죽어 실장육이 됨. 사육실장이 되면 행복이 100이 됨.
- **사료/성장(사료분배기 필수)**: 우리 안 개체는 **사료분배기 5×5 범위 안**에서만 분배기의 선택 사료(실장푸드/짓소산 푸드/운치)를 먹고 **성장 2배**. **사료분배기 범위 밖 개체는 우리 바닥의 운치(똥)를 먹으며 3% 확률로 육질이 떨어짐**. 운치는 항상 소모량×2 만큼 바닥에 배설(추출/얼룩은 아래 운치 항목). **자실장이 성체로 진화하면 30초간 새끼를 잡아먹지 않음.**
- **우리(생산탭, 가변 크기)**: **드래그로 크기 지정**해 설치. 1칸당 성체 5 / 새끼 10 수용, 초과분은 못 들어오고 공장 바닥을 **배회**. 우리 클릭→**이름 변경**(기본 N번 우리). 실장석은 우리 안에서 살아 돌아다니며, **성체가 근처 새끼를 잡아먹기도** 함(주변 새끼는 "테챠아!" 도망). 우리 안 개체는 밖으로 안 나가고, **밖을 배회하던 새끼가 우리에 들어오면 자동 수용**(성체는 우리에 못 들어감). 시작 시 좌측에 3×5 우리+성체 5마리.
- **물류**: 컨베이어(2/칸·자유 꺾임), **횡단벨트(3칸 다리)**, **터널(1×5: 입구→출구로 화물 순간 이동, 중간 건너뜀)**, 분류기(2칸·전방 출력), 집게(□→△·필터), 창고(3×3).
- **벨트**: 출력 방향이 안 막혔으면 무조건 밀어냄. 한 칸 2마리 초과로 들어오면 **옆으로 도망**(배회). 배회 실장석이 벨트에 올라가면 **자동 탑승**.
- **창고 = 저장**: 생산품은 즉시 안 팔리고 **재고 저장**(판매는 ②거래창). 실장푸드/운치는 자원 재고로.
- **입·출고**: 집게 □를 **우리**에 대면 생물을 필터로 꺼냄, **창고**에 대면 화물만 꺼냄. △로 우리/장치/벨트에 넣음.
- **거래(②)**: ⓐ실장푸드(사료) 10/50/100/1000개 구매(개당 ₩2), ⓑ**조미료 구매**(가격이 1분마다 ±1~10원 변동 — 조리실 재료), ⓒ**생물 구매**(성체실장 ₩120 / 자실장 ₩40 → 우리로, 가득 차면 배회), ⓓ**우리 실장석 판매**(사육실장 성체/새끼·독라 성체/새끼를 우리에서만 1/10/100/전부 판매; 배회 중은 못 팖), ⓔ**창고 재고 판매**(종류별 1/10/100/전량).
  - 사육실장 가격 = **개념 비례 + 크기 반비례**(작고 똑똑할수록 비쌈), 독라 가격 = **육질·크기 비례**. 새끼는 성체 계수 공유. 계수는 `config.js`의 `G.PRICE_DEFAULTS`. 판매 목록엔 **개당 가격**도 표시.
  - **판매금지**: 우리를 클릭해 패널의 **판매금지** 토글 → 그 우리의 실장석은 거래탭 판매 대상에서 제외.
- **생산**: 출산대(성체/독라→10초마다 **점액 덩어리**, "뎃데로게~"). 점액 덩어리는 **10초 뒤 구더기**로 변함(그 전 세면대 투입 시 구더기/엄지/자실장 분류). 사육실장만 출산대에서 새끼사육실장을 낳음. 출산대 **클릭=정보창**(장착 스탯·남은 수명·출산 수, 개체 제거 시 실장육).
- **가공**: 세면대(구더기·점액덩어리→구더기/엄지/자실장), **탈복기**(성체→독라 / 자실장→새끼독라 / 사육실장→독라·새끼사육실장→새끼독라), **도축기**(독라/새끼독라→실장육+위석 / **구더기·엄지→소형위석, 자실장→중형위석, 성체실장→대형위석**), **분쇄기**(실장석류·실장육→분쇄육, **소형위석→조미료1 / 중형→3 / 대형→5**), **배합기**(아래), **조리실(3×2)**.
- **배합기(3종 슬롯)**: **분쇄육1 + 운치10 → 실장푸드50** / **짓소산1 + 실장푸드50 → 짓소산 푸드50**(예전 조리실 레시피가 배합기로 이전). 일꾼 슬롯3.
- **조리실**: 재료+조미료1 → 실장육보다 비싼 요리. **구더기5=꼬치훈제 / 실장육2=통조림 / 독라1=진공포장 / 분쇄육2=실장젓갈 / 엄지·자실장3=실장무침**.
- **운치(배설물)**: 우리 안 실장석 수에 비례해 **우리 바닥에 누적**(우리당 최대 1000=녹색 얼룩 100개; 운치 10당 얼룩 1개). **집게에 운치 필터를 켜야 10개 묶음으로 추출**되고, 얼룩도 그만큼 사라짐. 추출한 운치를 **창고에 넣어야 운치 자원으로 누적**(배합기 사료 제작/사료분배기 급여에 사용). 즉, 우리를 방치하면 얼룩만 쌓이고 자원은 안 늘어남.
- **교정시설(3×3·6마리 수용)**: **자실장·성체실장**을 6마리까지 가둠. 일정 간격 대사("주인님께 반항하면…" 등)마다 **주사위: 50% 개념+1(교사 개념이 높으면 더 큼) / 50% 체력-1**. **체력 0 → 실장육**(뒤쪽 배출구), **개념 30↑ & 30초↑ → 졸업**(자실장→**새끼사육실장**, 성체실장→**사육실장**, 졸업 시 **행복 100**, 앞쪽 배출), **개념 20↓는 대사마다 30% 확률 탈출**.
- **출산대**: 부모 종류 무관(성체실장/독라/**사육실장 포함**) 모두 **점액 덩어리**를 낳음. **사육실장(성체)은 우리에서 오래 살수록 크기 증가(최대 100).**
- **자동판매(거래탭)**: 생산품마다 토글. 켜면 그 물자가 생기는 즉시 자동 판매(창고에 안 쌓임).
- **렌더 보강**: 기계에 장착·상호작용 중인 실장석은 아이콘이 아닌 본연 스프라이트로 표시. 우리는 `penbox_tile.png` 48×48 바닥 타일을 먼저 깔고, `penbox.png` 144×144 9분할 울타리를 비정형 셀 모양에 맞춰 그림. 아래쪽 경계는 실장석보다 나중에 다시 그려 앞 울타리처럼 보이게 함.
- **특수장치(1x1, 영향 범위)**: 태교 스피커(3×3 육질/개념/크기↑확률), 레드 포인터(1×5 출산 가속), 포획기(5×5 배회개체를 출력칸으로 모음·필터), 꼬챙이(**자실장·성체실장만** 고정 "테겍 테겍"·3×3 육질↑), **사료분배기**(연구 불필요·₩500·**우리 안에도 설치 가능**·5×5 급식·성장2배, 클릭→사료 선택/**모든 사료분배기에 적용**), **자동 포탑**(아래 침입/약탈 참고). 설치/고스트 시 범위 표시.
- **배회 음식 섭취**: 배회 중인 실장석은 아주 가끔 근처 벨트의 음식(실장푸드/실장육/분쇄육/요리)을 먹어치움 — **"우마우마한~"**(성체=데스/자실장=테치/엄지=레치/구더기=레후, 성장 상태별 어미). 수치는 `WANDER_EAT_CHANCE`.
- **외부 가족 침입(평화)**: 가끔(2.5~6분) 시작 필드 밖에서 실장석 가족(성체1+새끼1~3)이 리젠되어 안으로 기어들어옴 → 들어온 뒤엔 우리 없는 **배회 실장석**이 됨(노란 점선 링). 포획기/우리로 회수 가능(공짜 자원).
- **외부 침입 단계(0~4단계)**: 게임 시작 **20분 후부터 5~10분 간격**으로 **침입 무리**(빨간 점선 링, 머리 위 `침입 Lv.N` 표식)가 몰려옴. **규모는 첫 5마리에서 레이드마다 +3씩 증가, 최대 50마리(새끼 제외). 50 도달 후엔 수 대신 침입 HP가 레이드마다 +15%씩 증가.** **단계는 소유한 그리드 수**로 결정 — **1개=0 / 2개=1 / 4개=2 / 7개=3 / 10개=4단계**(소유 그리드 = 시작 필드 1 + 구매한 외부 그리드). 한 무리에 **0~현재단계 성체가 섞여** 옴(단계↑=육질·개념·크기·HP 강함, 성체는 `out_jisou1~5.png` 그래픽, 0단계·새끼는 기본 그래픽). **새끼는 0단계 성체와만 동행.** 침입 실장석은 **'침입' 태그**로 분류되고 **클릭/드래그·Del 불가**, 배회·벨트 위 **어린 새끼를 적극 공격**(잡아먹음)하고 벨트 음식도 약탈. **벨트에 먹을 게 없으면 새끼 든 우리로 침입**(우리 경계 무시). 충분히 먹거나 시간이 지나면 이탈. **벽은 막을 수 있음**(우리 경계와 달리 벽은 못 넘음).
- **자동 포탑(특수 탭, 연구 불필요·기본 제공, 설치 ₩1,000)**: `turret_tile.png`(하부 받침)+`turret.png`(상부 포신). 사거리(원형) 안 대상으로 **포신이 조준 회전하며 사격**, 적이 없으면 **천천히 회전**. **설치된 포탑 클릭 → 전용 업그레이드 창**: 공격력/연사/사거리(포탑별 개별, 비용 = 기본가×(레벨+1)) + 목표 설정(약탈자만/외부 출신/전부). 실장석 HP = 10+크기(+단계 보너스). **죽은 실장석은 실장육을 남김**. 노동석은 절대 쏘지 않음.
- **노동 교화소(생산 탭, 2×2, ₩800)**: **독라/새끼독라 투입 → 노동석**(청록 점선 링). 노동석은 플레이어 구역 안만 배회하고 **벨트·포획기·집게(긴팔집게 포함)에 안 잡히며 우리 경계 충돌도 무시**(우리를 가로질러 다님). **클릭 → 명령 창**: ⓐ**배회**(기본) ⓑ**회수** — 바닥 화물·배회 실장석(침입 제외)을 주워 **화물은 창고로, 실장석은 지정한 우리로**(다중 선택 필터 + 우리 지정) ⓒ**방어** — 침입자를 추격·공격(처치 시 실장육) ⓓ**대기** — **가장 가까운 노동 교화소 인근에서 머묾**. "모든 노동석에 적용" 버튼으로 일괄 명령. 여전히 독라라서 **도축기에 넣을 수 있고**(드래그/집게), **사료 재고를 먹으며 성장**(새끼독라 노동석→독라 노동석).
- **벽**: **Q** 또는 특수 탭의 벽 버튼으로 설치. 타일 모서리 점에서 시작해 직선으로 드래그하며, 벽이 있으면 배회 실장석이 그 경계를 넘지 못함.
- **일꾼 슬롯**: 세면대/교정시설/탈복기/배합기/도축기는 **독라·성체실장을 일꾼으로** 최대 3마리 받음(빈 슬롯은 네모 테두리 표시). 일꾼 1마리당 가공 **속도 +30%**. ※도축기엔 독라=도축대상/성체실장=일꾼, 탈복기엔 성체실장=가공대상/독라=일꾼처럼 **입력으로 쓸 수 있으면 가공, 아니면 일꾼**으로 자동 분기.
- **사육실장 / 독라(옛 노예석)** = 상품이자 실장석 → **우리에서만 살고 창고엔 못 들어감**. 성체 사육실장은 새끼를 낳음(출산대). 성체실장·독라는 포식자(기본 새끼 포식 + **사육실장 공격**), 사육실장은 피동(공격 안 함). 사육실장이 공격당해 **육질 0이 되면 "데챠앗!" 비명과 함께 독라**(새끼사육실장→새끼독라)가 됨 → 도축 가능.
- **등급**: 3스탯 중 최고값 기준 최하/하/중/상/특상. 스프라이트 위에 등급+스탯(육/개/크) 표기.
- **필터(분류기·집게)**: 품목 선택(구더기/엄지/자실장/성체실장/**사육실장 성체·새끼/독라 성체·새끼**/실장육/분쇄육/실장푸드/운치) + **스탯 조건(특정 스탯 이상/이하)**으로 분류.
- **배회 개체**: 장치를 제거하면 그 위의 생물이 공장 바닥을 돌아다님(성체 빠름→구더기 느림). 돌아다니다 조건 맞는 가공시설에 들어가면 자동 처리됨.
- **생물 직접 다루기**: 우리 안 생물은 살아 움직이고, 벨트 위/배회 중인 실장석을 **클릭=정보 / 드래그=집어 옮기기** 가능.
- 자원 순환 예: 우리 바닥 운치 → (집게 운치필터로 10묶음 추출) → 창고(운치 자원 누적) → 배합기(+분쇄육) → 실장푸드(사료) → 우리.
- **공원(①)**: 실장석이 자유 배회. **박스/실장푸드/물병**을 설치(돈 소모)하면 실장석이 가져가 **출산 확률↑**. 성체실장은 자실장/엄지/구더기를 낳고 — 자실장·엄지는 부모를 따라다니는 **가족**, 구더기는 **독립 배회**. 포획은 실장석을 **포획 상자로 드래그&드롭**. **포획 상자는 기본 1개 + 최대 4개 추가 구매**(₩600, 우측에 세로 배치). 각 상자의 **▾ 드롭다운으로 보낼 우리를 지정**(미지정=자동/가까운 우리, 가득 차면 공장 바닥 배회).
- **대사**: 출산 "뎃데로게~", 세면대 직후(구더기 "레후~"/엄지 "레치!"/자실장 "테치~!"), 벨트·배회 중 종별 멘트(말풍선).
- **렌더**: 새끼는 작게(자실장 2/3, 엄지·구더기 1/2), 겹치면 아래쪽이 위로(입체감), 개체 간 충돌 분리. 모든 이동속도 기본의 2/3.
- **연구(④) = 공원 업그레이드 5종**(레벨식, 비용 = 기본비용×(레벨+1)):
  - 등장 가중치 ↑ — **애호파 공원**(사육실장), **학대파 공원**(독라).
  - 야생 스탯 보너스(레벨당 +3~5) — **기초교육**(개념), **필라테스**(육질), **실장푸드 뿌리기**(크기).
