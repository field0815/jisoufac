# 실장석 공장 (Siljangseok Factory)

쿠키클리커 + 팩토리오 스타일 **물류 · 운송 · 사육 · 제작 · 판매** 게임.
주인공은 육류 판매업자. 실장석을 포획·번식·사육·가공·판매하여 돈을 번다.

## 실행 방법
`index.html` 더블클릭 또는 브라우저로 열기. (해상도 1440×960, 창에 맞춰 자동 축소) 시작 자금 ₩5,000.
※ `D:\OneDrive\게임제작\삼도호텔` 은 별개의 RPG Maker 프로젝트이니 혼동 주의.

## 폴더 구조
```
실장석공장/
├─ index.html
├─ css/style.css
├─ js/
│   ├─ config.js     ← 모든 수치/장치/가격계수/에셋 경로 (밸런스 조절 지점)
│   ├─ assets.js     ← 이미지/효과음 로더 (없으면 색상 플레이스홀더)
│   ├─ state.js      ← 게임 상태
│   ├─ creatures.js  ← 스탯/번식/성장/가격/생산품
│   ├─ pens.js       ← 우리(사료 소모·운치 배설·성장·수용량)
│   ├─ park.js       ← 공원(포획)
│   ├─ pen.js        ← 공장 좌측 "우리 도크"(성체 드래그 투입)
│   ├─ factory.js    ← 공장(그리드/벨트/장치/시뮬레이션) ★핵심
│   ├─ ui.js         ← 상단바 + 오버레이(거래/연구/통계) + 치트
│   └─ main.js       ← 초기화 + 게임 루프
└─ assets/images/{creatures,devices,backgrounds,ui}/ , assets/sounds/{sfx,bgm}/
```

## 그래픽 / 효과음 (스프라이트 시트만 넣으면 적용)
모든 그래픽은 **4프레임 애니메이션**으로 돌아갑니다. 각 `assets/...` 폴더의 `_여기에_넣으세요.txt`에 격자 규격이 있습니다.
- **실장석**: `creatures/` → adult/maggot/thumb/child .png — **4프레임 × 4방향(앞/좌/우/뒤)** = 4열×4행 시트. 이동 방향에 맞는 줄이 자동 선택됨.
- **장치**: `devices/` → belt, crossbelt, tunnel, sorter, grabber, warehouse, penbox(★9분할 울타리·무애니), birthing/**birthing_ready**(빈 상태), washbasin, slaughter, deshell, grinder, correction, mixer, cookery, 특수(speaker/pointer/catcher/skewer_dev/feeder) .png — **가로 4프레임** 시트.
- **화물(생산품/자원)**: `products/` → meat(실장육)/minced(분쇄육)/food(실장푸드)/unchi(운치)/skewer(꼬치훈제)/can(통조림)/vacuum(진공포장)/jeotgal(실장젓갈)/muchim(실장무침) .png — **정적 1장(애니메이션 없음)**. 아이콘이 곧 맵 위 화물 그래픽.
- 효과음: `sounds/sfx/` → capture, place, rotate, birth, wash, sell, click, remove, grow .mp3
- 프레임 속도는 `js/config.js`의 `ANIM_FPS`(기본 6). 이미지가 없으면 살짝 움직이는 색 플레이스홀더로 표시됩니다.

## 핵심 시스템
- **사료/성장**: 우리 안 개체만 사료(실장푸드) 분당 소모 — 성체5·자실장3·엄지2·구더기1. **운치는 먹지 않아도 항상 소모량×2 만큼 배설**. 사료가 공급되면 3분마다 성장(구더기→엄지→자실장→성체). **사료분배기 범위 안에서는 성장·사료소모 2배**.
- **우리(생산탭, 가변 크기)**: **드래그로 크기 지정**해 설치. 1칸당 성체 5 / 새끼 10 수용, 초과분은 못 들어오고 공장 바닥을 **배회**. 우리 클릭→**이름 변경**(기본 N번 우리). 실장석은 우리 안에서 살아 돌아다니며, **성체가 근처 새끼를 잡아먹기도** 함(주변 새끼는 "테챠아!" 도망). 우리 안 개체는 밖으로 안 나가고, **밖을 배회하던 새끼가 우리에 들어오면 자동 수용**(성체는 우리에 못 들어감). 시작 시 좌측에 3×5 우리+성체 5마리.
- **물류**: 컨베이어(2/칸·자유 꺾임), **횡단벨트(3칸 다리)**, **터널(1×5: 입구→출구로 화물 순간 이동, 중간 건너뜀)**, 분류기(2칸·전방 출력), 집게(□→△·필터), 창고(3×3).
- **벨트**: 출력 방향이 안 막혔으면 무조건 밀어냄. 한 칸 2마리 초과로 들어오면 **옆으로 도망**(배회). 배회 실장석이 벨트에 올라가면 **자동 탑승**.
- **창고 = 저장**: 생산품은 즉시 안 팔리고 **재고 저장**(판매는 ②거래창). 실장푸드/운치는 자원 재고로.
- **입·출고**: 집게 □를 **우리**에 대면 생물을 필터로 꺼냄, **창고**에 대면 화물만 꺼냄. △로 우리/장치/벨트에 넣음.
- **거래(②)**: ⓐ실장푸드(사료) 10/50/100/1000개 구매(개당 ₩2), ⓑ**조미료 구매**(가격이 1분마다 ±1~10원 변동 — 조리실 재료), ⓒ**생물 구매**(성체실장 ₩120 / 자실장 ₩40 → 우리로, 가득 차면 배회), ⓓ**우리 실장석 판매**(사육실장 성체/새끼·독라 성체/새끼를 우리에서만 1/10/100/전부 판매; 배회 중은 못 팖), ⓔ**창고 재고 판매**(종류별 1/10/100/전량).
  - 사육실장 가격 = **개념 비례 + 크기 반비례**(작고 똑똑할수록 비쌈), 독라 가격 = **육질·크기 비례**. 새끼는 성체 계수 공유. 계수는 `config.js`의 `G.PRICE_DEFAULTS`. 판매 목록엔 **개당 가격**도 표시.
  - **판매금지**: 우리를 클릭해 패널의 **판매금지** 토글 → 그 우리의 실장석은 거래탭 판매 대상에서 제외.
- **생산**: 출산대(성체/독라→10초마다 **점액 덩어리**, "뎃데로게~"). 점액 덩어리는 **10초 뒤 구더기**로 변함(그 전 세면대 투입 시 구더기/엄지/자실장 분류). 사육실장만 출산대에서 새끼사육실장을 낳음. 출산대 **클릭=정보창**(장착 스탯·남은 수명·출산 수, 개체 제거 시 실장육).
- **가공**: 세면대(구더기·점액덩어리→구더기/엄지/자실장), **탈복기**(성체→독라 / 자실장→새끼독라 / 사육실장→독라·새끼사육실장→새끼독라), 도축기(**독라 태그만**→실장육), 분쇄기(실장석류·실장육→분쇄육, **0.2초당 1마리**·무게100당1), **배합기(분쇄육1+운치10→실장푸드50)**, **조리실(3×2)**.
- **조리실**: 재료+조미료1 → 실장육보다 비싼 요리. **구더기5=꼬치훈제 / 실장육2=통조림 / 독라1=진공포장 / 분쇄육2=실장젓갈 / 엄지·자실장3=실장무침**.
- **교정시설(3×3·6마리 수용)**: **자실장·성체실장**을 6마리까지 가둠. 일정 간격 대사("주인님께 반항하면…" 등)마다 **육질-1·개념+1**. **육질 0 → 실장육**(뒤쪽 전용 배출구), **개념 30↑ & 30초↑ → 졸업**(자실장은 **새끼사육실장**, 성체실장은 **사육실장**, 앞쪽 배출), **개념 20↓는 대사마다 30% 확률 탈출**("똥닝겐은 스시와…").
- **출산대**: 부모 종류 무관(성체실장/독라/**사육실장 포함**) 모두 **점액 덩어리**를 낳음. **사육실장(성체)은 우리에서 오래 살수록 크기 증가(최대 100).**
- **자동판매(거래탭)**: 생산품마다 토글. 켜면 그 물자가 생기는 즉시 자동 판매(창고에 안 쌓임).
- **렌더 보강**: 기계에 장착·상호작용 중인 실장석은 아이콘이 아닌 본연 스프라이트로 표시. 우리(penbox)는 애니메이션 없이 144×144을 48×48 9분할한 **울타리 타일**로 그림.
- **특수장치(1x1, 영향 범위)**: 태교 스피커(3×3 육질/개념/크기↑확률), 레드 포인터(1×5 출산 가속), 포획기(5×5 배회개체를 출력칸으로 모음·필터), 꼬챙이(실장석 1마리 고정 "테겍 테겍"·3×3 육질↑), 사료분배기(3×5 성장2배·사료2배). 설치/고스트 시 범위 표시.
- **벽**: **Q** 또는 특수 탭의 벽 버튼으로 설치. 타일 모서리 점에서 시작해 직선으로 드래그하며, 벽이 있으면 배회 실장석이 그 경계를 넘지 못함.
- **일꾼 슬롯**: 세면대/교정시설/탈복기/배합기/도축기는 **독라·성체실장을 일꾼으로** 최대 3마리 받음(빈 슬롯은 네모 테두리 표시). 일꾼 1마리당 가공 **속도 +30%**. ※도축기엔 독라=도축대상/성체실장=일꾼, 탈복기엔 성체실장=가공대상/독라=일꾼처럼 **입력으로 쓸 수 있으면 가공, 아니면 일꾼**으로 자동 분기.
- **사육실장 / 독라(옛 노예석)** = 상품이자 실장석 → **우리에서만 살고 창고엔 못 들어감**. 성체 사육실장은 새끼를 낳음(출산대). 성체실장·독라는 포식자(기본 새끼 포식 + **사육실장 공격**), 사육실장은 피동(공격 안 함). 사육실장이 공격당해 **육질 0이 되면 "데챠앗!" 비명과 함께 독라**(새끼사육실장→새끼독라)가 됨 → 도축 가능.
- **등급**: 3스탯 중 최고값 기준 최하/하/중/상/특상. 스프라이트 위에 등급+스탯(육/개/크) 표기.
- **필터(분류기·집게)**: 품목 선택(구더기/엄지/자실장/성체실장/**사육실장 성체·새끼/독라 성체·새끼**/실장육/분쇄육/실장푸드/운치) + **스탯 조건(특정 스탯 이상/이하)**으로 분류.
- **배회 개체**: 장치를 제거하면 그 위의 생물이 공장 바닥을 돌아다님(성체 빠름→구더기 느림). 돌아다니다 조건 맞는 가공시설에 들어가면 자동 처리됨.
- **생물 직접 다루기**: 우리 안 생물은 살아 움직이고, 벨트 위/배회 중인 실장석을 **클릭=정보 / 드래그=집어 옮기기** 가능.
- 자원 순환 예: 우리→운치 → (창고서 집게로 추출) → 배합기(+분쇄육) → 실장푸드(사료) → 우리.
- **공원(①)**: 실장석이 자유 배회. **박스/실장푸드/물병**을 설치(돈 소모)하면 실장석이 가져가 **출산 확률↑**. 성체실장은 자실장/엄지/구더기를 낳고 — 자실장·엄지는 부모를 따라다니는 **가족**, 구더기는 **독립 배회**. 포획은 드래그&드롭, 자동 사냥은 종류 무관.
- **대사**: 출산 "뎃데로게~", 세면대 직후(구더기 "레후~"/엄지 "레치!"/자실장 "테치~!"), 벨트·배회 중 종별 멘트(말풍선).
- **렌더**: 새끼는 작게(자실장 2/3, 엄지·구더기 1/2), 겹치면 아래쪽이 위로(입체감), 개체 간 충돌 분리. 모든 이동속도 기본의 2/3.
- **연구(④) = 공원 업그레이드 5종**(레벨식, 비용 = 기본비용×(레벨+1)):
  - 등장 가중치 ↑ — **애호파 공원**(사육실장), **학대파 공원**(독라).
  - 야생 스탯 보너스(레벨당 +3~5) — **기초교육**(개념), **필라테스**(육질), **실장푸드 뿌리기**(크기).

- 그 밖 수치는 `G.CONFIG`.
